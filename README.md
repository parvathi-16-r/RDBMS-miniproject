# 🎬 MovieBook — Online Movie Ticket Booking

A web application that lets users browse movies, check showtimes, select seats, and book tickets online.

---

## 🚀 Features

- Browse now-playing and upcoming movies
- View showtimes by date, theater, and location
- Interactive seat selection
- User registration and login
- Booking history and e-ticket generation
- Payment integration
- Admin panel to manage movies and schedules

## 📦 Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/movie-booking-site.git
   cd movie-booking-site
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Set up environment variables**

   Create a `.env` file in the root directory:
   ```env
   PORT=3000
   DB_URI=your_database_connection_string
   JWT_SECRET=your_jwt_secret
   PAYMENT_API_KEY=your_payment_key
   ```

4. **Run the app**
   ```bash
   npm start
   ```

5. Open your browser and go to `http://localhost:3000`
